## Farenda — Task Manager
#### Особенности

- Modern UI: Полностью на Jetpack Compose с поддержкой Material Design 3.

- Persistence: Ваши задачи не пропадают после перезагрузки благодаря SharedPreferences.

- Clean Architecture: Четкое разделение логики и интерфейса (MVVM).

- Dynamic Input: Удобная панель добавления задач с опциональным описанием.

#### Технологический стек 

- Language: Kotlin

- UI Framework: Jetpack Compose

- Architecture: MVVM (ViewModel, LiveData)

- Local Storage: SharedPreferences + Gson (для сериализации объектов)

#### Установка

1. Перейдите в раздел Releases этого репозитория.

2. Скачайте файл farenda-v1.0.apk из последнего выпуска.

3. Откройте скачанный файл на вашем Android-устройстве.

4. Важно: Если система выдаст предупреждение, разрешите «Установку из неизвестных источников» в настройках браузера или проводника.
